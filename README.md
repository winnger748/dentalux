# 🦷 DentaLux — Премиальный Landing Page для стоматологии

Этот проект ориентирован на создание "доверительного" интерфейса. В медицине визуал напрямую влияет на конверсию, поэтому основной фокус здесь — чистота, премиальное восприятие и безупречная техническая реализация.

## 🛠 Процесс разработки (Iteration Process)
Проект создавался в несколько этапов, чтобы обеспечить максимальное качество:
1. **Архитектура контента:** Проработка блоков таким образом, чтобы информация о врачах и услугах считывалась мгновенно.
2. **Верстка и Стилизация:** Использование CSS Custom Variables для создания единой дизайн-системы. Это позволяет менять цветовую схему всего сайта за считанные секунды.
3. **Оптимизация:** Применение ИИ для глубокого анализа структуры DOM и минимизации вложенности тегов. Это сделало код более читаемым и производительным.

## 🚀 Технические особенности
* **Clean Architecture:** Семантический HTML5, который "понятен" поисковым роботам.
* **Modern CSS:** Активное использование CSS Grid для сложных сеток, которые в мобильной версии элегантно перестраиваются в стек.
* **Zero Dependencies:** Весь функционал (плавный скролл, липкое меню, анимации) написан на чистом JavaScript. Никаких JQuery или тяжелых плагинов.
* **Smart Reveal:** Анимации реализованы через `Intersection Observer`, что гарантирует плавность даже на слабых устройствах.

## 💡 Функциональные детали
* **Конверсионная форма:** Продуманная форма записи с микро-взаимодействиями, повышающими лояльность пользователя.
* **Sticky Header:** Динамическое изменение прозрачности и высоты меню при скролле для лучшей навигации.
* **Adaptive First:** Принцип адаптивности заложен на уровне архитектуры, а не через "быстрые фиксы" в конце.
