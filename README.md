# 🦷 DentaLux — Премиальный Landing Page для стоматологии

Этот проект ориентирован на создание "доверительного" интерфейса. В медицине визуал напрямую влияет на конверсию, поэтому основной фокус здесь — чистота, премиальное восприятие и безупречная техническая реализация.

## 🛠 Процесс разработки (Iteration Process)
Проект создавался в несколько этапов для достижения максимального качества:
1. **Архитектура контента:** Проработка блоков таким образом, чтобы информация о врачах и услугах считывалась мгновенно.
2. **Модульная сборка:** Разделение проекта на независимые слои (HTML, CSS, JS), что позволило выстроить логичную и чистую файловую структуру.
3. **SEO & Оптимизация:** Интеграция семантических тегов и Meta-данных (Open Graph) для корректного отображения сайта в поисковых системах и соцсетях.

## 🚀 Технические особенности
* **Modular Codebase:** Весь код разнесен по соответствующим файлам (`style.css`, `script.js`). Это исключает путаницу и соответствует современным стандартам разработки.
* **Modern CSS:** Использование CSS Custom Variables для управления дизайн-системой и CSS Grid для создания сложных адаптивных сеток.
* **Zero Dependencies:** Весь функционал (плавный скролл, липкое меню, анимации) написан на чистом JavaScript. Никаких тяжелых библиотек.
* **Smart Reveal:** Анимации реализованы через `Intersection Observer API`, что гарантирует высокую производительность и плавность 60 FPS.

## 💡 Функциональные детали
* **SEO Optimized:** Полный набор Meta-тегов для продвижения клиники в поисковой выдаче.
* **Sticky Header:** Динамическое изменение прозрачности и высоты меню при скролле для удобной навигации.
* **Adaptive First:** Интерфейс спроектирован так, чтобы идеально выглядеть на экранах любого размера — от мобильных до 4K.
