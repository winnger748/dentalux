# 🦷 DentaLux — Landing Page стоматологической клиники премиум-класса

Современный, адаптивный лендинг для стоматологической клиники, ориентированный на высокую конверсию и премиальное визуальное восприятие. Проект демонстрирует навыки создания чистого интерфейса, плавной анимации и оптимизированного кода.

## 🚀 Стек технологий
* **HTML5** — семантическая структура.
* **CSS3 (Custom Variables)** — использование переменных для гибкого управления стилями и темой.
* **JavaScript (Native)** — реализация интерактивных элементов без использования тяжелых библиотек.
* **Intersection Observer API** — современные анимации появления элементов при скролле.
* **CSS Grid & Flexbox** — надежная и гибкая верстка.

## 💡 Особенности проекта
* **Premium UI/UX:** Дизайн выполнен в стиле минимализма с акцентом на типографику и пространство (whitespace).
* **Full Responsiveness:** Сайт идеально отображается на десктопах, планшетах и смартфонах.
* **Performance:** Высокая скорость загрузки благодаря отсутствию лишних зависимостей.
* **AI-Enhanced Workflow:** При создании проекта использовались инструменты искусственного интеллекта для оптимизации структуры кода и повышения эффективности разработки.

## 🛠 Функционал
* Продуманная форма записи на прием с валидацией полей.
* Динамическая смена стилей хедера при прокрутке.
* Адаптивное мобильное меню (Burger Menu).
* Плавные визуальные эффекты (Reveal animations) для повышения вовлеченности пользователя.
